# Heart Disease Prediction

## Overview
This project predicts the presence of heart disease using Machine Learning algorithms.

## Dataset
Heart Disease Dataset

## Algorithms Used
- Logistic Regression
- Naive Bayes
- SVM
- KNN
- Decision Tree
- Random Forest
- XGBoost
- Neural Network

## Best Result
Random Forest Accuracy: 90.16%
